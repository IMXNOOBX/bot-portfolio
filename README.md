## üöÄ Getting Started

* Development
```bash
git clone https://github.com/IMXNOOBX/bot-portfolio.git
cd bot-portfolio
npm install
npm run dev
```

* Production
```bash
npm run build
```
> The build will be in the `build` folder and will be statically generated.

* Docker
```bash
docker build -t bot-portfolio .
docker run -d -p 80:80 bot-portfolio
```
